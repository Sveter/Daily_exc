#### 题目描述: 字符串反转
写出一个程序，接受一个字符串，然后输出该字符串反转后的字符串。例如：

输入描述:
输入N个字符

输出描述:
输出该字符串反转后的字符串

示例1
输入

abcd
输出

dcba

#### 题目标题：公共字串计算

计算两个字符串的最大公共字串的长度，字符不区分大小写

详细描述：

接口说明

原型：

int getCommonStrLength(char * pFirstStr, char * pSecondStr);

输入参数：

     char * pFirstStr //第一个字符串

     char * pSecondStr//第二个字符串

样例输入
asdfas werasdfaswer

样例输出
6
